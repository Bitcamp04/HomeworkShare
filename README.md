## Github 프로젝트 Merge 방법
1. 개인 브런치로 변경하여 개인 브런치에 다른 사람의 브런치들을 모두 merge 시킨다.
2. main 브런치로 변경하여 merge 시킨 브런치를 다시 merge 시킨다.
